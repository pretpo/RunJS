# RunJS

1) Encode your JavaScript source using`LZString.compressToEncodedURIComponent(...)`.
2) Open `/#<your-generated-uri>` (paste the encoded payload after`/#`).

Note: For security, external assets can’t be loaded (no remote scripts, styles, images, etc.).
